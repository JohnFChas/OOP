# Labb 11 - Events
## Uppgifter
* Skriv ett program där användaren kan skriva in heltal
* Skapa följande event och delegat:
	* Event: NumberInput
	* Delegat: AnalyzeNumber
* Skapa följande metoder för delegaten AnalyzeNumber:
	* IsEven
	* IsDivisableByThree
	* IsPrimeNumber
* Lägg till tre instanser av AnalyzeNumber i NumberInput (en för varje metod)
* Varje gång användaren skriver in ett nummer ska programmet höja eventet och skriva ut lämpligt meddelande för varje metod
