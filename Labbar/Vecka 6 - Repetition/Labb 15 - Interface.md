# Labb 15 - Interface
## Uppgifter
* Skapa följande interface med metoder:
	* IPushable
		* Push()
	* IVehicle
		* Start()
		* Stop()
		* Lock()
		* Unlock()
* Skapa följande klasser och låt dem implemetera interface:
	* Button : IPushable
	* Boulder : IPushable
	* Car : IVehicle
	* Bicycle : IVehicle
	* Spaceship : IVehicle
* Låt varje klass definiera funktionalitet för de interface de implementerar
